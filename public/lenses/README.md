# Lens photos — INSTALLED

Real transparent-PNG lens photos are in use (from Sebastien's Drive), wired via the
`image` field in `lib/vcpManifest.ts`:

| File | Lens |
|------|------|
| `c-series.png` | Panavision C-Series (anamorphic) |
| `signature-prime.png` | ARRI Signature Prime (modern spherical) |
| `zoom.png` | Modern Zoom (modern spherical zoom) |
| `cooke.png` | Cooke Speed Panchro (vintage) |

To replace one, just overwrite the file (keep the name). If a file is missing, the
drum falls back to the original SVG glyph automatically.
