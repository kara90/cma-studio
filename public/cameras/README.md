# Camera photos (optional)

Same as lenses — drop a photo here and it replaces the SVG glyph in the Body drum
automatically (falls back to the glyph if missing).

Expected filenames → camera:

| File | Camera |
|------|--------|
| `alexa-35.png` | ARRI ALEXA 35 |
| `alexa-lf.png` | ARRI ALEXA LF |
| `red-raptor.png` | RED V-RAPTOR XL |
| `panaflex.png` | Panavision Panaflex Gold |

Then set the `image` field for that camera in `lib/vcpManifest.ts`.

⚠️ Same copyright note as lenses — use your own or licensed photos, not
manufacturer marketing shots.
